## Name

## Description
This project was created as part of the Oasis Infobyte Graphic Design Internship Program.

The objective of this task was to create a creative and visually appealing magazine layout using images, typography, and design elements. The magazine was designed to provide an engaging reading experience while maintaining a clean and aesthetic appearance.

- Modern and aesthetic design
